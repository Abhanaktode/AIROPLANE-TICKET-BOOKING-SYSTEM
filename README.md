
# ✈️ Airport Management System

A desktop-based Airport Management System developed using Python, Tkinter, and MySQL to manage flight operations and passenger bookings efficiently.

## 🚀 Features

* Admin Login Authentication
* Add and Manage Flights
* View Available Flights
* Real-Time Seat Availability
* Passenger Seat Booking System
* Automatic Seat Status Updates
* MySQL Database Integration
* User-Friendly GUI using Tkinter & ttk

## 🛠️ Technologies Used

* Python
* Tkinter
* MySQL
* ttk Widgets

## 📌 Project Overview

This project provides a simple and interactive airport management solution where admins can add flights and passengers can view flights, check available seats, and book tickets. The system automatically updates seat availability and stores all booking details in the MySQL database.

## 💡 Learning Outcomes

* GUI development using Tkinter
* Database connectivity with MySQL
* CRUD operations
* Real-time data handling
* Backend logic implementation in Python
